## 使用 npm 安装
推荐使用 npm 的方式进行开发，享受 node 生态圈和 webpack 工具链带来的便利。<br/>
通过 npm 安装的模块包，我们可以轻松的使用 import 或者 require 的方式引用

<br/>

```bash
npm install gov-ui --S
```
