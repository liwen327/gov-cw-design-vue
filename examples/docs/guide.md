# 介绍

----

`gov-ui` 是一款基于 `Vue.js 2.0` 的前端 UI 组件库

## 特性

- 基于 `Vue` 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES6
- 提供友好的 API，可灵活的使用组件
